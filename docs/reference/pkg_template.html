Start a new project with the cpp4r package template

Description

The template includes a DESCRIPTION file, a minimal R/ directory and placeholders with instructions. You can then edit these files to customize your new package.

Arguments

pathPath to the new project
pkgnameName of the new package

Return

The file path to the copied template (invisibly).

Examples

# create a new directory
dir <- tempdir()
dir.create(dir)
## [1] FALSE
# copy the package template into the directory
pkg_template(dir, "mynewpkg")
## [1] "/tmp/RtmpGIjIBg"
